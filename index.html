CPT foundation

Page 2 • CPT foundation lab

Learn the frame first: identify what CPT is, what a modifier does, and how to keep the base code separate from the extra story added after it.

Teach + check Fast feedback

What CPT does

CPT is the main code set used to describe procedures and services. Think of the base code as the main service story.

What a modifier does

A modifier does not replace the base code. It adds a detail about circumstance, component, repetition, laterality, distinctness, or relationship.

What students must notice

Tiny wording shifts matter: same physician, another physician, same day, post-op period, reduced, discontinued.

Recognize each level fast

Straightforward
Minimal problem burden, minimal/no data, minimal risk.
Low
Limited complexity, limited data, low risk.
Moderate
More involved problem work, more analysis, and/or prescription or management risk.
High
Very serious complexity, threat to health, and/or high-risk decisions.
Rule to remember: 2 of 3 must meet the level

Do not let one high element trick you. One high element by itself does not make the whole visit high.

Low
Limited problem complexity, limited data, low risk.
Moderate
Multiple or more involved problems, moderate data and/or prescription or management risk.
High
Serious threat / high complexity, extensive thinking burden, and/or high risk decisions.
